# msc-thesis-agh
Repository containing all resources from my Master's Thesis at AGH-UST.

Contents:
* ```dnn-mode-connectivity``` - clone of https://github.com/timgaripov/dnn-mode-connectivity with applied changes to make it run
* ```dnn-mode-connectivity-tf``` - custom implementation of FGE in TensorFlow
* ```latex``` - source of my thesis in latex
* ```workshop``` - materials from the Master Thesis Workshop 2020/2021
* ```materials.md``` - list of interesting resources encountered during research
